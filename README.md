# Ecommerce-Store-CI-CD

## Technologies used
- This project was created using HTML, CSS, JavaScript, React, Typescript, React Query, React bootstrap, Firebase, bootstrap,CI/CD, and Redux. 

## Installation & setup
1. Click the link to open up my GitHub page. 

https://github.com/clounsbu/Ecommerce-Store-CI-CD

2. Once on the GitHub page click on the **green** button to the right that says **code**. This will bring a dopdown menu, you will click on **HTTPS** and copy the link below it. 
3. Open you terminal in VS code and type
```sh
git clone https://github.com/clounsbu/Ecommerce-Store-CI-CD.git
```
4. You will see a folder named **Ecommerce-Store-CI-CD**. Open the folder in **Visual Studio Code**.
5. To view the project open the terminal and type npm run dev and the app will be uploaded.


## How to use as a typical user
- Once your on the website you will see a navbar with Home, Register, and Login.
- Underneath the navbar you will see Grey Ghost Shopping Center and to the right a button that says cart with the number 0 next to it. 
- Below the title you will see a green button to add a product and if you click it a form will open at the top of the page to add a new product. 
- Below that you will see a drop down to the far left that says all products. 
- If you click on the drop down you can select from all the different categories to view those specific items. 
- Below the product category drop down you will see cards containing all of the products with their title, image, description, rating and price. 
- You will also see a button at the bottom of each card that says add to cart, edit, and delete. 
- Click the add to cart button to add the items to the cart. You can add multiple of the same item as well.
- Click the edit button to edit any of the products on the page and the delete button to delete an item. 
- To view your cart go back up to the navbar and select the cart button.
- After clicking the cart button you will be brought to the cart page. 
- On the cart page you will see a button at the top labeled continue shopping that will bring you back to the products. 
- Below that will be small cards with the items you added to the cart. If you have multiple of the same item you will only see that item once but the quantity will represent how many of that item are in the cart.
- Each card also has a remove button at the bottom and if you click that the item will be removed from the cart. 
- At the very bottom you will see the number of total items and the total price. 
-When you are ready to checkout click the checkout button and you will get a message saying checkout successful and the cart will be emptied.
- If you click the register button you will be taken to the register page. There you will fill out a form with your information including name, email, and password. After you submit the form you will be brought to you profile page. 
- The profile page you will see a welcome message with your name with your name and email under it. You can edit your nme there and if you want to change your name click the blue button underneath that says update profile. 
- You will also see a yellow button that allows you to view any of your orders and a red button next to that allows you to delete your profile. 
- You will also notice after you register or login the nav menu changes so it now includes Home, Profile, and logout. 
- If you want to logout hit the logout button and you will be brought back to the home page.
